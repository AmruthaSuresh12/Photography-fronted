# 📸 Photography Frontend

A simple and elegant frontend project built with **HTML & CSS**, designed to showcase photography with a clean, user-friendly interface.

---

## ✨ Features
- Minimalist layout for better focus on images  
- Responsive design for mobile, tablet, and desktop  
- Smooth hover effects and transitions  
- Grid-based gallery for easy navigation  

---
